# Legacy Root Markdown Archive

This folder contains markdown files moved from the frontend repo root during documentation cleanup.

## Purpose

- Reduce top-level noise.
- Keep active docs easy to locate.
- Preserve historical implementation notes without deleting them.

## Usage

- Treat files here as historical context.
- Do not treat archived files as canonical source-of-truth unless explicitly re-promoted.
- If a legacy file is still relevant, migrate its valid parts into active docs and keep this archive untouched.
